API Security Risk Analysis

## Overview

This project analyzes common API security risks using public testing APIs and Postman.

## Tools Used

* Postman
* JSONPlaceholder
* ReqRes API
* GitHub

## Security Risks Analyzed

* No Authentication
* Excessive Data Exposure
* Missing Rate Limiting
* Information Disclosure

## Repository Structure

* screenshots/
* report/
* requests/

## Ethical Notice

This project was conducted only on public test APIs intended for learning purposes. No unauthorized systems were targeted.
